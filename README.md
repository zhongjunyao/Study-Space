## vue-dev 源码文件目录说明
```
vue-dev
|-- benchmarks  用于存放基准测试文件
|-- dist        用于存放打包出来的 vue 框架文件，内有各种版本的vue
|-- examples    用于存放 vuejs 使用案例文件
|-- flow        用于存放 flow 规则配置文件
|-- pakages     用于存放引用的框架包
|-- scripts     用于存放pakage.json中使用到的各种执行文件
|-- src         用于存放 vue 框架项目的代码资源
|-- test        用于存放测试用例
|-- types       用于存放编译 typescript 的配置文件及与 typescript 相关的内容

```
